# MOUSE AI IPTV + Chat
واجهة بث IPTV مع دردشة مباشرة باستخدام Firebase.

## المميزات
- بث HLS مباشر
- قائمة قنوات من ملفات M3U
- دردشة مباشرة باسم مستعار
- تصميم متجاوب لجميع الأجهزة
- حقوق محفوظة أسفل الصفحة

## كيف تستخدمه
1. افتح `index.html` في المتصفح.
2. اختر قناة من القائمة أو استخدم الروابط السريعة.
3. دردش مع المستخدمين الآخرين باسم مستعار.

## رفع على Vercel
1. ارفع المشروع على GitHub.
2. اربطه بحساب Vercel.
3. اختر المشروع للتوزيع وسيعمل مباشرة.
